## Règles documents EcoVadis (rappel)

- **Nom/logo CAMELEON COM MARK SASU** sur tout document
- **Date de parution** obligatoire, ≥ 1 mois avant remise
- **Validité** : Politiques/Actions 8 ans · KPI/audits RSE 2 ans · Certificats date d'expiration (ou 3 ans)
- **55 documents max** par évaluation
- Format PDF privilégié, max 30 Mo, pas de mot de passe
- Pas de docs combinés (sauf certifs multi-sites, KPI consolidés, manuels EHS)
- Pas de docs purement légaux ni ISO 9001 seul

Voir [METHODOLOGIE-ECOVADIS.md](METHODOLOGIE-ECOVADIS.md) pour le détail.
