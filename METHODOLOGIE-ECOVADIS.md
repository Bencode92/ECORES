# Méthodologie EcoVadis — Règles documents

Synthèse opérationnelle du *Guide sur les documents EcoVadis* (V4 FR, 2024).

## Système de gestion RSE — 3 piliers · 7 indicateurs

```
Politiques  → POLI (politiques)
            → ENDO (adhésions)

Actions     → MESU (mesures)
            → CERT (certifications)
            → COVE (couverture / déploiement)

Résultats   → REPO (reporting KPI)
            → 360° (veille externe)
```

## 4 thèmes · 21 critères RSE

1. **Environnement** — énergie/GES, eau, biodiversité, déchets, pollution, produits, etc.
2. **Social et Droits Humains** — S&S, conditions travail, formation, discrimination, travail forcé/enfants
3. **Éthique** — corruption, fraude, conflits d'intérêts, sécurité de l'information
4. **Achats Responsables** — politique, due diligence, évaluation, audits, formation acheteurs

## Exigences communes à TOUS les documents

| Exigence | Détail |
|---|---|
| Nom/logo | **CAMELEON COM MARK SASU** doit apparaître. Pour Excel/captures : domaine email cameleons-rh.fr ou note de confidentialité avec nom |
| Date de parution | Visible et ≥ 1 mois avant remise |
| Lisibilité | Pas de mot de passe, pas corrompu, texte modifiable de préférence |
| Format | PDF privilégié. Accepté : .docx, .xls, .pdf, .ppt, .odt, .jpeg, .png. Max 30 Mo |
| Lien à la réponse | Doit être rattaché au moins à une question (sinon ignoré) |

## Délais de validité

| Type | Durée |
|---|---|
| Politiques | 8 ans |
| Actions | 8 ans |
| Reporting KPI quantitatifs (couverture incluse) | 2 ans |
| Rapports d'audit RSE externe | 2 ans |
| Certificats | date d'expiration (ou 3 ans si non précisée) |
| Conventions collectives | date d'expiration éventuelle |
| Adhésions | pas d'expiration mais vérifié à chaque éval |

## Documents NON valides

- Sans nom/logo de l'entreprise
- Strictement juridiques ou émis par autorité publique seule
- ISO 9001 seul, qualité, continuité d'activité, info financière sans section RSE
- Philanthropie / bénévolat externe
- Documents combinés (sauf : certifs multi-sites, KPI consolidés, manuels EHS, plans formation+matériel+registres)
- Auto-attestations rédigées pour le seul questionnaire EcoVadis
- Plagiat, certificats falsifiés
- Hyperliens (sauf rapport RSE en ligne sur site officiel, avec sections indiquées)

## Limite de 55 documents

Privilégier les documents **transversaux** qui couvrent plusieurs questions :
- Rapport RSE annuel
- Code de conduite / éthique global
- Manuel EHS
- Politiques consolidées par thème

## Bibliothèque à 4 catégories

Pour chaque question, viser au moins :
1. **Une politique** (engagement, objectif chiffré + année cible)
2. **Une action** (procédure, formation, équipement, certification)
3. **Un KPI de reporting** (chiffre, unité, année, comparatif)
4. Et, si possible, **une preuve de couverture** (% sites/employés couverts)

## Checklist pré-soumission (rappel guide p.19)

- [ ] Périmètre d'évaluation compris et documents alignés
- [ ] Tous documents avec nom CAMELEON, dates récentes, exigences techniques
- [ ] Documents prioritaires transversaux téléchargés en premier
- [ ] Chaque réponse sélectionnée liée à au moins un document
- [ ] Pages indiquées pour chaque document multi-pages
- [ ] Validité des documents 2025 (réutilisation) vérifiée
- [ ] PAC (Plan d'Actions Correctives) consulté et lié si pertinent
